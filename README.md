# Instagram Knowledge Base

Интерактивный сайт для хранения, организации и быстрого поиска ваших полезных постов из Instagram.

## Быстрый старт (Локально)
1. `npm install`
2. `npm run dev`

---

## 🤖 Как добавлять новые посты с помощью ИИ

Чтобы добавить новый пост в вашу базу знаний, вы можете отправить любому искусственному интеллекту (например, ChatGPT или Claude) следующую инструкцию:

**Скопируйте и отправьте этот Промпт в ИИ:**

```text
Я хочу добавить новый пост из Instagram в свою базу знаний. 
Вот ссылка и текст поста: 
[ВСТАВЬТЕ ССЫЛКУ И ТЕКСТ СЮДА]

Пожалуйста, проанализируй текст и выдай мне ТОЛЬКО один валидный JSON объект (без markdown-оберток и лишнего текста) в следующем формате:
{
  "url": "Ссылка на пост",
  "description": "Оригинальный текст поста (очищенный от мусора)",
  "author": "Имя автора (если есть, иначе пусто)",
  "category": "Выбери одну из: 'Искусственный Интеллект (AI)', 'Программирование и Технологии', 'Офис и Продуктивность', 'Языки и Образование', 'Бизнес Финансы и Маркетинг', 'Здоровье и Разное'",
  "triggerWord": "Если автор просит написать в комментариях какое-то кодовое слово для получения ссылки/материала, напиши это слово. Если нет - null",
  "tools": ["Список упомянутых инструментов: Claude, Excel, Word, Python, Figma и т.д. Если нет - пустой массив"],
  "hasList": true/false (если в тексте есть пошаговая инструкция или список),
  "links": ["любые внешние http-ссылки найденные в тексте. Если нет - пустой массив"]
}
```

**После того как ИИ выдаст вам JSON:**
1. Зайдите на ваш сайт.
2. Нажмите кнопку **"Добавить пост"**.
3. Вставьте полученный JSON в окно и нажмите **"Импортировать"**.
4. Пост моментально появится в нужной категории!

## Деплой на Netlify

Этот проект готов к развертыванию на Netlify:
1. Запушьте код в GitHub.
2. В Netlify выберите этот репозиторий.
3. Нажмите "Deploy". `netlify.toml` уже настроен.
